如何切换主题

hexo官方文档并没有说明要如何切换主题，这里mark一下

  1. 找到自己想要切换的主题的github仓库地址
  2. 在themes目录下执行git clone命令
  3. 在_config.yml中找到theme配置项，按照github仓库的指引进行配置
  4. 重启hexo server
